## 何ができるの？

**tabularize** は、散らかったブラウザを整理するための Chrome 拡張です。  
90日分の閲覧履歴を自動で集計し、よく見るサイトをひとめで把握。開きっぱなしのタブを、ドメインごとにワンクリックでグループ化します。

---

## 機能

### 📊 URL・ドメイン集計
- 直近 **90日分の閲覧履歴**を自動取得・集計
- **URL一覧**：タイトル・URL・訪問回数を表示。訪問数順 / 最終訪問日順でソート可能
- **ドメイン一覧**：ドメインごとの合計訪問回数を表示
- インクリメンタル検索（URL・タイトル・ドメイン名に対応）
- 行クリックで既存タブにフォーカス、なければ新規タブを開く

### 🗂 タブグループ化
- **ドメイン単位でグループ化**：任意のドメインのタブをワンクリックでまとめる
- **全タブを一括グループ化**：開いているすべてのタブをドメインごとに整理。アクティブなグループ以外は自動で折り畳み
- **常時グループ化**：指定したドメインのタブは開くたびに自動でグループに追加
- **グループ解除**：ドメイン単位でグループを解除

### ↕️ タブ自動ソート
- **アルファベット順ソート**をON/OFFでトグル
- ONにすると新しいタブが開かれるたびに自動で並び替え（ピン留めタブは除外）

### 🚫 除外機能
- 特定ドメインを集計対象から除外（除外後は既存データも即時削除）

## インストール

Chrome Web Store には現在公開していません。開発者モードで手動インストールしてください。

```bash
git clone https://github.com/satou30/tabularize.git
cd tabularize
```

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をオン
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. クローンしたリポジトリの **`src/`** フォルダを選択

---

## 使い方

1. Chrome ツールバーの tabularize アイコンをクリック → サイドパネルが開く
2. **URL一覧** でよく見るページを確認・クリックしてジャンプ
3. **ドメイン一覧** で「グループ化」ボタンを押すとそのドメインのタブが一箇所にまとまる
4. 「常時▶」ボタンをONにすると、以降そのドメインのタブは自動でグループ化される
5. 「タブ並び替え」をONにすると全タブがアルファベット順に自動整列

## 必要なパーミッション

| パーミッション | 用途 |
|---|---|
| `history` | 閲覧履歴の読み取り・集計 |
| `tabs` | タブ情報の取得・操作 |
| `tabGroups` | タブグループの作成・更新・解除 |
| `storage` | 集計データの永続化 |
| `alarms` | 90日クリーンアップの定期実行 |
| `sidePanel` | サイドパネルUIの表示 |

> ホスト権限 (`host_permissions`) は**使用しません**。`history` パーミッションにより全URL対象でAPIが動作するため不要です。

## 技術スタック

- **Vanilla JavaScript** — フレームワーク・バンドラー不使用
- **Plain CSS** — プリプロセッサ不使用
- **Biome** — Lint & Format
- **Chrome APIs** — `chrome.history`, `chrome.tabs`, `chrome.tabGroups`, `chrome.storage`, `chrome.alarms`, `chrome.sidePanel`

---

## 開発

```bash
npm install          # Biome をインストール
npm run lint         # Lint チェック
npm run format       # 自動整形
```

変更後は `chrome://extensions` の「更新」ボタンで拡張を再読み込みしてください。

- **background.js のデバッグ**：`chrome://extensions` → 「Service Worker」リンク → DevTools
- **サイドパネルのデバッグ**：サイドパネル上で右クリック → 「検証」
